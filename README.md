# McMaster MATH 1MP3 Winter 2026 

This repository contains Notebooks and other course material for MATH 1MP3 Winter 2026 sections C01 and C02.

## "Magic link" to upgrade jupyter files

Click on [this link](https://mcmaster.syzygy.ca/jupyter/hub/user-redirect/git-pull?repo=https%3A%2F%2Fgithub.com%2Fbourdin%2FMATH1MP3_W2026.git&urlpath=lab%2Ftree%2FMATH1MP3_W2026.git%2FNotebooks&branch=main) before every lecture to automatically update the course material on http://mcmaster.syzygy.ca.

For the technically inclined and those familiar with [git](https://git-scm.com/), this link pulls files from the course [git repository](https://github.com/bourdin/MATH1MP3_W2026.git).
